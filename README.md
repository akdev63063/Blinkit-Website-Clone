# Blinkit-Website-Clone
A responsive front-end clone of the Blinkit grocery delivery website built using only HTML and CSS. This project focuses on recreating the UI/UX design of a modern quick-commerce platform.
.

🚀 Features
🛒 Clean and modern homepage design
📦 Product listing layout
🔍 Search bar UI
🧭 Navigation bar with categories
📱 Fully responsive design (mobile + desktop)
🎨 Attractive UI inspired by Blinkit
🧑‍💻 Tech Stack
HTML5 – Structure of the website
CSS3 – Styling and layout
Flexbox & Grid – Responsive design
📂 Project Objective

The goal of this project is to:

Practice frontend development basics
Improve layout and design skills
Understand how real-world websites like Blinkit are structured
📸 Pages Included
Home Page
Product Section
Category Section
Footer

git clone https://github.com/your-username/blinkit-clone.git
cd blinkit-clone

index.html
